# 📱 Garage Cabinet — Інструкція запуску з нуля

## Крок 1 — Встановіть Node.js

Завантажте та встановіть з https://nodejs.org (версія LTS)

Перевірте у терміналі:
```
node --version   → має показати v20.x або вище
npm --version    → має показати 10.x або вище
```

---

## Крок 2 — Встановіть Expo CLI

```bash
npm install -g expo-cli eas-cli
```

---

## Крок 3 — Розпакуйте проект

Скопіюйте папку `garage-app` у зручне місце, наприклад:
```
C:\Projects\garage-app\
```

---

## Крок 4 — Встановіть залежності

```bash
cd C:\Projects\garage-app
npm install
```

---

## Крок 5 — Запустіть у браузері (швидкий тест)

```bash
npx expo start --web
```

Відкриється браузер з додатком. Натисніть `W` у терміналі якщо не відкрилось.

---

## Крок 6 — Переглянути на реальному Android-телефоні

1. Встановіть на телефон **Expo Go** з Google Play
2. Запустіть у терміналі:
   ```bash
   npx expo start
   ```
3. Відскануйте QR-код камерою телефону (або в додатку Expo Go)
4. Додаток завантажиться на телефон ✅

> ⚠️ Комп'ютер і телефон мають бути в **одній Wi-Fi мережі**

---

## Крок 7 — Підключення до реального 1С

### 7.1. Налаштування на стороні 1С

Програміст 1С має зробити:

1. Відкрити конфігуратор → Регламентні операції → HTTP-сервіси
2. Додати новий HTTP-сервіс з ім'ям `gsk`
3. Базовий URL: `/gsk/hs`
4. Створити методи (GET):
   - `/owner` — дані власника по `garageNumber`
   - `/balance` — баланс
   - `/charges` — нарахування
   - `/payments` — платежі
   - `/electricity` — лічильник
   - `/notices` — оголошення
5. Метод POST:
   - `/electricity/submit` — передача показань

**Формати відповідей описані в коментарях файлу `src/services/api1C.js`**

### 7.2. Налаштування мережі

Варіант А (локальна мережа):
- Телефон і сервер 1С в одній мережі
- BASE_URL = `http://192.168.X.X:PORT/gsk/hs`

Варіант Б (через інтернет):
- Налаштуйте VPN або port forwarding на роутері
- Або використайте ngrok для тестування: `ngrok http 8080`

### 7.3. Увімкніть реальний API в коді

Відкрийте `src/services/useGarageData.js`:
```js
// Змініть цей рядок:
const USE_MOCK = true;
// На:
const USE_MOCK = false;
```

Вкажіть реальні дані у `src/services/api1C.js`:
```js
BASE_URL: 'http://192.168.1.100:8080/gsk/hs',
USERNAME: 'ваш_логін_1с',
PASSWORD: 'ваш_пароль_1с',
```

## Крок 8 — Зібрати APK для Android (без Google Play)

1. Зареєструйтесь на https://expo.dev (безкоштовно)
2. Виконайте:
   ```bash
   eas login
   eas build --platform android --profile preview
   ```
3. Через ~15 хвилин отримаєте посилання на скачування APK
4. Встановіть APK на телефон (дозвольте установку з невідомих джерел)

## Структура файлів

```
garage-app/
├── App.js                          ← Головний файл (всі екрани)
├── package.json                    ← Залежності
└── src/
    ├── constants/
    │   └── index.js                ← Кольори + mock дані
    └── services/
        ├── api1C.js                ← API методи для 1С
        └── useGarageData.js        ← Хук завантаження даних
```

## Проблеми та рішення

| Проблема | Рішення |
|----------|---------|
| "Network request failed" | Перевірте IP-адресу сервера 1С |
| "CORS error" | Налаштуйте CORS у HTTP-сервісі 1С або використайте proxy |
| Expo Go не бачить QR | Переконайтесь що в одній Wi-Fi мережі |
| APK не встановлюється | Дозвольте установку з невідомих джерел у налаштуваннях |
